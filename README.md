# Cisco Networking Labs (CCNA)

## Overview
Hands-on networking labs simulating real-world enterprise environments using Cisco devices.

## Topics Covered
- VLAN configuration
- Inter-VLAN routing
- OSPF / IPv6 routing
- ACLs and network segmentation
- NAT and firewall basics

## Tools
- Cisco Packet Tracer
- Real Cisco hardware

## Example Labs
- Multi-router OSPF topology
- VLAN segmentation with trunking
- IPv6 routing configuration

## Skills Demonstrated
- Network troubleshooting
- CLI configuration
- Routing & switching
